ВАЖНО
Некоторые прототипы исходного LaTeX используют внешние файлы изображений, которых не было среди загрузок. Для них сохранены изображения-заглушки с точными исходными именами файлов. Перечень находится в файле НЕДОСТАЮЩИЕ_КАРТИНКИ_ЗАДАНИЯ_3.txt.
